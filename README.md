# 📊 E-Commerce Sales Dashboard (Microsoft Excel)

## 📌 Project Overview
This project is an interactive E-Commerce Sales Dashboard built using Microsoft Excel. It provides insights into sales performance through Pivot Tables, Pivot Charts, KPIs, and Slicers, allowing users to analyze business data efficiently.

## 🎯 Objectives
- Analyze overall sales performance
- Track key business metrics
- Compare sales across categories and cities
- Enable interactive data filtering using slicers

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📈 Dashboard Features
- Total Revenue
- Total Orders
- Average Order Value
- Category-wise Sales Analysis
- City-wise Sales Analysis
- Monthly Sales Trends
- Interactive Slicers for filtering
